# 아이콘 파일 위치

이 폴더에 다음 아이콘 파일들을 추가해주세요:

## 필수 파일

- `icon.png` - 256x256 이상의 PNG 파일 (Linux/일반용)
- `icon.ico` - Windows용 아이콘 파일
- `icon.icns` - macOS용 아이콘 파일

## 아이콘 생성 방법

1. 256x256 이상의 PNG 이미지를 준비합니다.
2. 온라인 변환기를 사용하여 ICO, ICNS 파일을 생성합니다.
   - https://cloudconvert.com/png-to-ico
   - https://cloudconvert.com/png-to-icns

## 임시 아이콘 없이 빌드

아이콘 파일이 없어도 빌드는 가능하지만, 기본 Electron 아이콘이 사용됩니다.
